Projects

Stock Ticker Tracker


Built a web app using Angular and PHP where users can log in, select certain stock tickers, and view stock charts and related news for those stocks. Users can also view global news for the day. Source Code available on GitHub.

UVA Craigslist Clone


Used the Python Django framework to build a website where students can buy and sell items to other UVA students, such as school-related gear or textbooks. Students login using their school credentials, and there is a map functionality to specify where the trade will take place. Source Code available on GitHub.

Traffic Stop Modeling and Prediction


Trained and tested machine learning models to predict if an individual would be stopped for a traffic violation in Virginia to observe if there is a racial bias among officers. Analyzed using k-means clustering, Logistic regression, SGD, and ultimately a custom neural network using TensorFlow and Keras. Source Code available on GitHub.

Rowing Game


An infinite runner game made with Java where the player must avoid obstacles to get as far as possible. It was made to practice Java and game design. Source Code available on GitHub.

Personal Website


This website was made using HTML, CSS, and JavaScript. It was made made it to practice responsive web design and GitHub. Source Code available on GitHub.

GPA Calculator and Planner


This application can calculate your current GPA based on grade and credits for individual classes or an entire semester. It can also tell you a required GPA for future classes based on a target GPA. Source Code available on GitHub.
